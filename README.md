## Tech Stack

**finTrack** is built with a powerful combination of technologies:

### Frontend

* **React:** The heart of the UI, providing a dynamic and engaging experience.
* **Vite:** Lightning-fast builds for rapid development.
* **TailwindCSS:** Elegant and responsive styling, making finTrack look amazing on any device.
* **Zustand:** Simplified state management for a smooth and efficient user experience.
* **Chart.js:** Interactive and visually appealing charts to visualize your financial data.

### Backend

* **Express:** A robust and scalable Node.js framework.
* **PostgreSQL:** A reliable and powerful database to keep your financial data secure.
* **JWT:** JSON Web Tokens for secure authentication.
* **Firebase:** Real-time data synchronization and other awesome features.

## Installation

Ready to take control of your finances? Here's how to get started:

1.  Clone the repository:

    ```bash
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    ```

2.  Navigate to the project directory:

    ```bash
    cd your-repo-name
    ```

3.  Install dependencies:

    ```bash
    npm install
    ```

4.  Set up your environment variables (refer to `.env.example` for the required variables).

5.  Run the application:

    ```bash
    npm run dev
    ```
